# Composants UI

SoloFlow utilise une bibliothèque de composants basée sur [shadcn/ui](https://ui.shadcn.com/) et [Radix UI](https://www.radix-ui.com/) pour offrir une interface moderne et accessible.

## Installation

Les composants sont déjà configurés dans SoloFlow. Pour ajouter de nouveaux composants :

```bash
npx shadcn-ui@latest add [component-name]
```

## Composants disponibles

### Boutons

```tsx
import { Button } from "@/components/ui/button";

// Variantes
<Button variant="default">Défaut</Button>
<Button variant="destructive">Destructeur</Button>
<Button variant="outline">Contour</Button>
<Button variant="secondary">Secondaire</Button>
<Button variant="ghost">Fantôme</Button>
<Button variant="link">Lien</Button>

// Tailles
<Button size="default">Défaut</Button>
<Button size="sm">Petit</Button>
<Button size="lg">Grand</Button>
<Button size="icon">Icône</Button>
```

### Cartes

```tsx
import {
  Card,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle,
```

```tsx
    <Label htmlFor="message">Message</Label>
```

## Thématisation

### Variables CSS

Les composants utilisent des variables CSS pour la thématisation :

```css
:root {
  --background: 0 0% 100%;
  --foreground: 222.2 84% 4.9%;
  --primary: 222.2 47.4% 11.2%;
  --primary-foreground: 210 40% 98%;
  --secondary: 210 40% 96%;
  --secondary-foreground: 222.2 84% 4.9%;
  /* ... */
}
```

### Mode sombre

Le mode sombre est automatiquement géré :

```css
.dark {
  --background: 222.2 84% 4.9%;
  --foreground: 210 40% 98%;
  /* ... */
}
```

### Personnalisation

Pour personnaliser un composant :

```tsx
import { cn } from "@/lib/utils";

<Button 
  className={cn(
    "bg-gradient-to-r from-purple-500 to-pink-500",
    "hover:from-purple-600 hover:to-pink-600"
  )}
>
  Bouton personnalisé
</Button>
```

## Icônes

SoloFlow utilise [Lucide React](https://lucide.dev/) pour les icônes :

```tsx
import { 
  Home, 
  User, 
  Settings, 
  ChevronDown,
  Plus,
  X 
} from "lucide-react";

<Button>
  <Plus className="mr-2 h-4 w-4" />
  Ajouter
</Button>
```

## Animations

Les composants incluent des animations fluides avec Framer Motion :

```tsx
import { motion } from "framer-motion";

<motion.div
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.3 }}
>
  <Card>Contenu animé</Card>
</motion.div>
```

## Accessibilité

Tous les composants respectent les standards d'accessibilité :

- **Navigation au clavier** complète
- **Lecteurs d'écran** supportés
- **Contraste** conforme WCAG 2.1
- **Focus** visible et logique
- **ARIA** labels appropriés

## Bonnes pratiques

### Structure

```tsx
// ✅ Bon
<Card>
  <CardHeader>
    <CardTitle>Titre</CardTitle>
  </CardHeader>
  <CardContent>
    Contenu
  </CardContent>
</Card>

// ❌ Éviter
<div className="card">
  <h3>Titre</h3>
  <p>Contenu</p>
</div>
```

### Responsive

```tsx
// ✅ Responsive
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  {items.map(item => (
    <Card key={item.id}>{/* ... */}</Card>
  ))}
</div>
```

### Performance

```tsx
// ✅ Lazy loading pour les composants lourds
const HeavyComponent = lazy(() => import('./HeavyComponent'));

<Suspense fallback={<div>Chargement...</div>}>
  <HeavyComponent />
</Suspense>
```
